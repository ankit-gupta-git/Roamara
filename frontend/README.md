# Roamara

Roamara is an Airbnb-like full-stack platform, a global online marketplace connecting travelers with hosts offering unique accommodations (like homes, apartments, treehouses) and local experiences.

## Technologies Used
- **MongoDB**
- **Express.js**
- **React.js**
- **Node.js**

## Features
- Developed a full-stack Airbnb-like platform with property listings, booking workflows, and secure authentication.
- Implemented role-based access and REST APIs, supporting booking flows for 50+ users during development and testing.

## Getting Started
To get a local copy up and running follow these simple steps.

### Prerequisites
- Node.js
- npm

### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/your_username/Roamara.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the server
   ```sh
   npm start
   ```

## Access
- **Frontend**: Runs on `http://localhost:3000` by default.
- **Backend**: Runs on `http://localhost:8080` (ensure backend is started separately).
- For production, deploy to your preferred hosting service (e.g., Vercel for frontend, Heroku for backend).
